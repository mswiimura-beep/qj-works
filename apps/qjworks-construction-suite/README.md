# QJworks 施工管理一気通貫デモ

中小工務店向けの施工管理Webアプリの商品サンプルです。

## 想定する流れ

1. 案件登録
2. 見積作成
3. 工程・段取り管理
4. 現場写真管理
5. 日報・完了報告
6. 請求・入金確認

## 使い方

`index.html` をブラウザで開きます。

このデモはサーバーなしで動きます。入力内容はブラウザ内に保存されます。

## 主な機能

- 案件、顧客、現場住所、担当、工程の管理
- 今日の作業、遅れ、報告待ち、未請求の確認
- 月間カレンダーで着工、完了、作業予定を確認
- 今日と明日の天気を右下に固定表示
- 施工前・施工中・施工後の写真登録
- 写真撮影とファイル選択の切り替え
- 写真の自動圧縮
- 日報入力と顧客向け報告文の作成
- 見積明細、請求済み、入金済みの管理
- JSON形式でのバックアップ書き出し・読み込み
- スマホのホーム画面追加に対応するmanifest

## 操作設計

- ホーム画面に大きな主要操作ボタンを配置
- 「案件」ではなく「現場」など現場担当者に伝わりやすい言葉を使用
- 現在選んでいる現場名をホームに大きく表示
- ボタンとチェックボックスを大きめに設定
- スマホ縦画面では1列表示を基本にする

## デザイン方針

- 紫からピンクのグラデーションは使わない
- テクノ・フューチャー風の装飾は使わない
- 単色、罫線、余白で業務画面として見せる
- 影は使わず、標準グリッドに沿って整列する
